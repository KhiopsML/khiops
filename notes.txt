Il faut compiler notre propre MPI et le packager avec Khiops
1. git clone du repo
c'est compliqué à mettre en place car CMake n'est pas utilisé
2. Utilisation de vcpkg
Il n'y a que OpenMPI
```bash
brew install vcpkg
git clone https://github.com/microsoft/vcpkg "$HOME/vcpkg"
export VCPKG_ROOT="$HOME/vcpkg"
cd khiops
vcpkg new --application
vcpkg add --help
vcpkg add port openmpi
cmake --fresh --preset macos-clang-release -DMPI=ON -DCMAKE_TOOLCHAIN_FILE=$VCPKG_ROOT/scripts/buildsystems/vcpkg.cmake
```
